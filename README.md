# Seafarer Realty — Website Prototype

A static, GitHub Pages–ready prototype for Seafarer Realty (Authentic Florida Keys Living),
built from the client's own brand guide and photography.

## Structure
- `index.html` — homepage
- `communities/` — 5 community pages (Key Largo, Islamorada, Marathon, Big Pine Key, Key West)
- `listings/` — listing search results page + 4 sample listing detail pages
- `blog/` — journal index + 3 sample articles
- `privacy-policy.html`, `accessibility.html`, `realtor-code-of-ethics.html`
- `assets/css/style.css` — shared stylesheet (brand tokens: Seafarer Green, Coastal Gold, Ocean Blue)
- `assets/js/main.js` — shared interactivity (nav, reveal animations, FAQ accordions, search)
- `assets/img/` — client-provided photography, optimized for web

## Notes
This is a design/content prototype, not a production build. FLEXMLS/IDX integration,
real listing data, and Alexia's headshot are placeholders pending final client assets.

## Local preview
Just open `index.html` in a browser — no build step required.

## Deploy
Push to GitHub and enable GitHub Pages (Settings → Pages → Deploy from branch → `main` → `/root`).
